# tunagirisken.dev

This repository contains the source code for **tunagirisken.dev**, my personal technical blog.

The blog focuses on embedded systems, real-time software, and low-level engineering. It is used to document practical experience, design notes, and in-depth technical articles.

---

## About

This project serves as a personal knowledge base and technical writing platform. The primary goals are to share real-world engineering experience, provide clear technical explanations, and build a long-term reference for embedded and systems programming topics.

Covered areas include embedded systems, RTOS, Time-Sensitive Networking (TSN), Linux and low-level system programming, networking protocols, and general software engineering practices.

---

## Tech Stack

The website is built using a static site generator and deployed via GitHub Pages. Continuous integration and deployment are handled using GitHub Actions.
